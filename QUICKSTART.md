## 📊 知识库规划

你需要准备的知识库数据：

```
quotes.json (初版 50-100 条)
├── 电影台词 (20-30 条)
├── 网络梗 (10-20 条)
├── 名人名言 (10-20 条)
└── 古诗词 (10-20 条)

每条需要包含:
{
  "quote": "金句内容",
  "source": "出处",
  "type": "电影/梗/名言/古诗",
  "category": "搞笑/深刻/温暖/破冰",
  "context": "适用场景",
  "author": "作者/角色"
}
```

## 💡 关键要点速记

1. **不追求数量，追求质量**  
   → 50 条精选金句 > 500 条垃圾数据

2. **场景驱动设计**  
   → 不同场景需要不同的知识库和 Prompt

3. **用户反馈很重要**  
   → 记录用户选择，根据反馈优化

4. **社区参与很关键**  
   → 从一开始就设计好贡献机制

5. **长期学习价值**  
   → ChatBuff 不止给建议，还要帮助用户成长
